
Help for @iluxa

### How to start

- `git clone git@github.com:lbvf50mobile/noko_test.git nkdir`
- `cd nkdir`
- `bundler install`
- `alias x="bundler exec ruby run.rb`
- `x` now **x** start **ruby run.rb** throught the **bundler** environment.

### History of commands
